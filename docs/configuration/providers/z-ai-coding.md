# Z.ai Coding Subscription

Z.ai offers a dedicated coding subscription plan with a separate API endpoint.

## Configuration

```json
{
	"name": "Z.ai Coding Subscription",
	"baseUrl": "https://api.z.ai/api/coding/paas/v4/",
	"apiKey": "your-z.ai-coding-api-key",
	"models": ["your-model-name"]
}
```

## Setup

1. Subscribe to the Z.ai coding plan
2. Use the coding-specific API key from your subscription
