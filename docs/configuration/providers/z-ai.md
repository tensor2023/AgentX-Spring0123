# Z.ai

[Z.ai](https://z.ai) (Zhipu AI) provides GLM language models through an OpenAI-compatible API.

## Configuration

```json
{
	"name": "Z.ai",
	"baseUrl": "https://api.z.ai/api/paas/v4/",
	"apiKey": "your-z.ai-api-key",
	"models": ["your-model-name"]
}
```

## Setup

1. Create an account at [z.ai](https://z.ai)
2. Generate an API key from the developer console
