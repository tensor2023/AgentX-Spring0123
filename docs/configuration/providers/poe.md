# Poe

[Poe](https://poe.com) provides access to multiple AI models through a single API.

## Configuration

```json
{
	"name": "Poe",
	"baseUrl": "https://api.poe.com/v1",
	"apiKey": "your-poe-api-key",
	"models": ["your-model-name"]
}
```

## Setup

1. Get an API key from [poe.com/api_key](https://poe.com/api_key)
2. A Poe subscription may be required for API access
